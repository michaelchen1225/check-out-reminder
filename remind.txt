We are building a Windows background utility called "Checkout Reminder" in Python.
The project is at: C:\Users\pinyu\OneDrive\桌面\Lndata\project\check-out-reminder

== What it does ==
- When the app launches (auto-starts with Windows login), it records the current time as "work start time"
- Work end time = start time + configurable work hours (default 9 hours)
- At end time, it shows a Windows balloon notification + a tkinter modal popup telling the user to check out
- If the app restarts mid-day, it restores the original start time from disk (doesn't reset the clock)

== Stack ==
- Python 3.10 at C:\Users\pinyu\AppData\Local\Programs\Python\Python310\
- pywin32 (already installed) — system tray icon + balloon notifications
- tkinter (stdlib) — modal dialog fallback
- winreg (stdlib) — Registry auto-start toggle
- PyInstaller — packages into dist\checkout_reminder.exe

== Files created ==
- checkout_reminder.py     main app (tray icon, timer, settings, notifications)
- build.bat                runs PyInstaller to produce dist\checkout_reminder.exe
- install_startup.bat      creates shortcut in Windows Startup folder
- uninstall_startup.bat    removes startup shortcut
- requirements.txt         pyinstaller>=5.0
- .gitignore               excludes dist/, build/, __pycache__

== Settings ==
